# The-Liberty-Way
Travel agency, that performs flights, and hotel reservations made in PHP.
Implementación de API’s Amadeus, Google Login, Aviationstack, RestCountries, Google Translate.
Organización, asignación y entregas del equipo de trabajo atraves del uso de metodología Agile.
Manejo de servidores (Raspberry Pi - Debian, Google VM) durante el desarrollo y lanzamiento del sitio.
Excelente experiencia usuario, con adaptación a telefonos móviles, tablets, etc.
Proyecto realizado a escala internacional
